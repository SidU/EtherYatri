Instructions for Installing EtherYatri.NET Visual Studio.NET Integration Module

 

NOTE: MS_VS.NET_INSTALLATION: denotes the folder where Visual Studio .NET is installed. On most machines it is ‘C:\Program Files\Microsoft Visual Studio .NET’.

 

 

1.      Copy folder ‘CSharpEtherYatriWiz’ to %MS_VS.NET_INSTALLATION%\VC#\VC#Wizards

 

2.      Copy 'EtherYatri Mobile Agent Application.*' to

          %MS_VS.NET_INSTALLATION%\VC#\CSharpProjects

 

3.      Copy ‘EtherYatri Mobile Agent.*’ to

          %MS_VS.NET_INSTALLATION%\VC#\CSharpProjectItems\LocalProjectItems

 

4.      Copy folder ‘CSharpEtherYatriAddAgentWiz’ to

%MS_VS.NET_INSTALLATION%\VC#\VC#Wizards

 

5.      Copy 'EtherYatri.dll' to

            %MS_VS.NET_INSTALLATION%\Common7\IDE\PublicAssemblies

 

6.      Use gactutil –I to register EtherYatri.dll.
