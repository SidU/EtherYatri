EtherYatri.IAgentHost.AgentExistsOnRemoteHost Function
AgentExistsOnRemoteHost is used to check whether the specified agent exists at the specified host.

Public bool AgentExistsOnRemoteHost (string, EtherYatri.AgentId)
  Type Name Description  
    string remoteHostAddress the url of the host where agent should be searched for  
    AgentId agentId the id of the agent to be searched for  
  Return Description  
    bool true if the specified agent exists at the specified host, false otherwise
