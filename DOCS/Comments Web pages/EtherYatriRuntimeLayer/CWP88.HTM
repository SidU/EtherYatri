EtherYatri.IAgentHost Interface
IAgentHost interface describes the methods which the EtherYatri.NET communication-layer must implement.

Access: Public
  Members Description  
    Execute Execute is invoked by remote hosts to transfer an agent to the local host.

 
    CreateAgent CreateAgent is called by remote hosts to create a new agent on local host.

 
    CreateAgentOnRemoteHost CreateAgentOnRemoteHost is used to create a new agent on a remote host.

 
    MoveAgent MoveAgent transfers the specified agent from the localhost to a remote host.

 
    MoveAgentOnRemoteHost MoveAgentOnRemoteHost moves an agent located on a remote host to another remote host.

 
    DisposeAgent Performs cleanup at the end of an agent's execution.

 
    DisposeAgentAtRemoteHost Used to dispose an agent located at a remote host.

 
    InvokeAgentMethod Used by remote hosts to invoke a method of an agent located at the local host.

 
    InvokeAgentMethodOnRemoteHost To invoke a method of an agent located at a remote host.

 
    AgentExists Remote hosts call AgentExists to determine whether specified agent exists on the local host.

 
    AgentExistsOnRemoteHost AgentExistsOnRemoteHost is used to check whether the specified agent exists at the specified host.
