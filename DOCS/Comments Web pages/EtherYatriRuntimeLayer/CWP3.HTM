EtherYatri.Agent Class
To create their own agents, clients must inherit from Agent class and override atleast its Run method. Additionally, if the agent would be moving from one host to another, it must be marked with Serializable attribute.

Access: Public
Base Classes: Object
  Members Description  
    agentManager A reference to the agent mananager who controls this agent.

 
    id The unique id associated with this agent.

 
    Run An Agent does its main work in Run method. Run is called on every host where the agent begins its execution.

 
    OnCreation OnCreation is fired when this Agent is created. OnCreation is fired only once during the entire lifetime of the agent.

 
    OnStopping OnStopping is fired when this Agent is stopped. OnStopping is fired if the agent is asked to terminate by its proxy.

 
    OnArrival Fired when the agent reaches a new host.

 
    OnMigration Fired just before the agent moves to another host.

 
    MoveTo Moves this Agent to another host.

 
    Initialize Used by EtherYatri.NET Framework to internally initialize the Agent

 
    Id The globally unique id associated with this agent.

 
    AgentContext The Agent's context.

 
    Proxy A proxy to the agent.

 
    AssemblyPath Path of the assembly containing the agent's code.
