HelloWorldAgent Demo

(C#)

 

Shows you how to write a simple agent that when loaded prints a Hello World message to the console and terminates.

 

Click here to open the zipped folder containing the solution.

 

 

Communication Demo

(C#)

 

Shows you how easy it is to write mobile agents with EtherYatri.NET that communicate with each other. Agent just needs to mark its method with AgentWebMethod attribute to make it accessible to peer agents.

 

Click here to open the zipped folder containing the solution.

 

 

MovingAgent Demo            

(C#)

 

Shows you how to write an agent that when loaded at an agent-host starts execution and then moves to another host.
