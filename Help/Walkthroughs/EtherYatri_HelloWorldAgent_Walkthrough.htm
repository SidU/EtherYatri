Walkthrough: Creating an EtherYatri HelloWorld mobile agent

 

This walkthrough guides you through the steps of creating your first mobile agent using EtherYatri.NET. The walkthrough demonstrates the rapidity with which the users can create mobile agent applications using the toolkit.

 

Create a new EtherYatri Mobile Agent Application

1.       Open Visual Studio .NET and on the File menu, click New, and then click Project.

2.       In the Project Types pane, click Visual C# projects, and in the Templates pane, click the EtherYatri Mobile Agent Application Icon.

3.       In the Name box, enter EtherYatriDemo.

4.       In the Location box, enter C:.

5.       Click OK to close the dialog box and create the project. A snapshot is shown below.

 

 

 

A new agent named Agent1 will be added to your project as shown in the snapshot below

 

 

 

Build the project

Press Control-Shift-B to build the project.

 

Load the agent in EtherYatri Windows Agent Host

Make sure the EtherYatri Windows Agent Host is running. Perform the following steps to load the agent into EtherYatri Windows Agent Host.

 

1.       Switch to EtherYatri Windows Agent Host application and on the Agent menu, click Create New Agent.

2.       In the Create New Agent dialog box, In the Path of agent’s assembly file box, enter C:\EtherYatriDemo\bin\Debug\EtherYatriDemo.dll.

3.       In the Full name of agent’s type box, enter EtherYatriDemo.Agent1.

4.       Press OK to close the Create New Agent window. A snapshot is shown below.

 

 

 

 

Verify output

The EtherYatri Agent Host’s output window should now display the agent’s Hello World! message, as shown in the snapshot below.

 

                                                                                   

 

 

You have just created your first mobile agent using EtherYatri.NET.
